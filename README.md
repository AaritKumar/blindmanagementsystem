# Accessible QR Audio Generator

BMS (Blind Management System) is a web application that creates QR codes accessible to visually impaired users by turning them into an audio-first experience.

Using BMS, users can scan QR codes generated by businesses that directly play a chosen spoken description.

## Problem

For visually impaired people, everyday tasks such as reading a menu, understanding a product label, or accessing public information can be frustrating and impossible. Completing these tasks requires visually impaired people to ask others for help, which increases dependence and makes simple tasks more bothersome.

As the world increasingly uses visual interfaces to communicate information, visually impaired individuals are often excluded and overlooked. Tools that are meant to increase convenience, such as digital labels or quick-access information systems, use visual elements and require sight to use, unintentionally excluding people who can't see.

This gap in accessibility places visually impaired people in a world that is not designed with them in mind.

BMS removes these barriers.

With the Blind Management System, visually impaired individuals no longer need to rely on sight to access basic information such as reading a menu or a product label. Users can open BMS and directly scan a QR code that speaks out clear spoken instructions.

This allows users to:

- Access information independently without needing assistance
- Understand products, menus, and public information in real time
- Avoid the frustration of navigating complex or inaccessible interfaces
- Feel more confident and included in everyday environments

BMS replaces the increasing dependency on visual elements with direct audio output and offers visually impaired people access to information previously unavailable.

## Key Feature: Directional Audio Scanner

The directional audio scanner is an audio beacon that guides users to the QR code:

- When the QR code is detected, a beep is played
- Sound shifts to the left or right based on the direction of the QR code
- Beep speed increases as users get closer to the QR code

## Quick Start

1. Clone the repository

```bash
git clone https://github.com/AaritKumar/blindmanagementsystem
cd blindmanagementsystem
```

2. Set up environment

```bash
python -m venv venv
source venv/bin/activate  # Mac/Linux
venv\Scripts\activate     # Windows
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run migrations

```bash
python manage.py migrate
```

5. Start the server

```bash
python manage.py runserver
```

6. Open the app

Go to: http://127.0.0.1:8000/

## How It Works

Businesses:

- Create products on the business dashboard
- Add descriptions of products onto the QR code
- Generate QR codes
- Organize QR codes for varying use cases

Users:

- Open the scanner
- Use the audio beacon to identify and scan a QR code
- Hear the audio description automatically

## Core Features

- Secure, login-protected dashboard
- Product and folder management
- Drag-and-drop organization
- Reusable description templates
- Automatic QR code generation
- Directional audio QR scanner

## Technical Overview

- Backend: Django (MVT architecture)
- Frontend: Modern JavaScript (ES6 classes)
- Audio System: Web Audio API (real-time spatial sound)
- Database: Django ORM with migrations
- Accessibility: ARIA roles and live regions

## Design Philosophy

The project was built with a fundamental principle:

> BMS should be more than a software, it should be a lifestyle.


## Summary

This project demonstrates how thoughtful design and engineering can solve real-world accessibility problems. By combining a QR code-based app with spatial audio and text-to-speech, it creates a system that is practical not only for visually impaired users but also for businesses and communities.
